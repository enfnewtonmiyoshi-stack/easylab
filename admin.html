ℹ️ Como Usar

1. Chave Admin: A chave padrão é sua_chave_secreta_aqui. Mude no arquivo server.js!

2. Adicionar Cliente: Digite chave + usuário + senha e clique "Criar Usuário"

3. Deletar Cliente: Clique no botão "Deletar" ao lado do usuário

4. O cliente usa: Abra easylab_producao.html e faça login com o usuário/senha que você criou
